# AI Profile Photo

React + Vite 기반의 웹 프로젝트입니다.

## 시작하기

### 의존성 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

개발 서버가 실행되면 브라우저에서 `http://localhost:5173`으로 접속할 수 있습니다.

### 프로덕션 빌드

```bash
npm run build
```

빌드된 파일은 `dist` 디렉토리에 생성됩니다.

### 빌드 결과 미리보기

```bash
npm run preview
```

## 기술 스택

- React 18
- Vite 5
- JavaScript (JSX)
